# Task 4 - Windows Firewall Configuration

## Objective
Configure and test Windows Firewall by creating and removing firewall rules.

## Steps Performed

1. Opened Windows Defender Firewall with Advanced Security.
2. Viewed existing inbound and outbound rules.
3. Created a new inbound rule to block TCP Port 23 (Telnet).
4. Verified that the rule was successfully created.
5. Deleted the firewall rule to restore the original configuration.

## Outcome

Successfully learned how to:
- View firewall rules
- Create firewall rules
- Block a network port
- Remove firewall rules
- Understand basic traffic filtering

## Tools Used

- Windows Defender Firewall with Advanced Security
- Windows 11
